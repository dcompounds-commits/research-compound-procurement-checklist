## The 8-Point Supplier Verification Checklist

Before placing any research compound order, verify all of the following:

### ✅ 1. Third-Party Certificate of Analysis (COA)
- [ ] COA issued by a **named, independent** testing laboratory
- [ ] Identity confirmed by **NMR spectrum** (¹H minimum)
- [ ] Identity confirmed by **mass spectrometry** (GC-MS or LC-MS)
- [ ] HPLC purity chromatogram included or available on request
- [ ] COA is **batch-specific** — not a generic document

### ✅ 2. Purity Standard
- [ ] Purity stated as **≥98% by HPLC area normalization**
- [ ] Method specified (area normalization, not "content" or "assay")
- [ ] Chromatogram available showing peak shape and baseline

### ✅ 3. Water Content
- [ ] Karl Fischer titration result present on COA
- [ ] Water content **<1%** for HCl salt forms
- [ ] Result is batch-specific, not a generic specification

### ✅ 4. Lot Traceability
- [ ] Unique lot/batch number on physical container label
- [ ] Lot number **matches** the accompanying COA
- [ ] COA dated within 24 months of purchase

### ✅ 5. Packaging Standards
- [ ] **Amber glass vials** or HDPE containers (not polystyrene)
- [ ] Desiccant included for hygroscopic compounds
- [ ] Sealed with inert closures
- [ ] UV-protected packaging for photosensitive compounds

### ✅ 6. Supplier Transparency
- [ ] Registered business name verifiable
- [ ] Country of manufacture/operation stated
- [ ] Research use only declaration on invoice and labeling
- [ ] Regulatory compliance documentation available on request

### ✅ 7. Technical Responsiveness
- [ ] Supplier answers specific pre-purchase technical queries
- [ ] Can name the third-party testing laboratory used
- [ ] Can provide solubility data across common lab solvents
- [ ] Response time acceptable for institutional procurement timelines

### ✅ 8. Quality Guarantee
- [ ] Written quality guarantee covering identity and purity
- [ ] Defined replacement or credit process for QC failures
- [ ] Policy applies to results from your own in-house testing

## COA Field Reference

| COA Field | Minimum Standard | Red Flag |
|---|---|---|
| Identity method | NMR + MS | Single method only |
| Purity | ≥98% HPLC area norm. | No method stated |
| Water content | <1% Karl Fischer | Not reported |
| Lot number | Batch-specific | Generic/undated |
| Testing lab | Named third-party | In-house only |
